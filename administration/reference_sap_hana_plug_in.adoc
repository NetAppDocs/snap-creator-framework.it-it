= Plug-in SAP HANA
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Il plug-in SAP HANA consente di creare backup ed eseguire il ripristino point-in-time dei database SAP HANA in base alle copie Snapshot dello storage.

Questo plug-in utilizza il client SAP HANA hdbsql per eseguire comandi SQL per garantire la coerenza del database e gestire il catalogo di backup SAP HANA. Il plug-in Ã¨ supportato per i programmi SAP Certified hardware Appliances e TDI (Tailored Datacenter Integration).

Il plug-in viene installato come parte di Snap Creator Agent su un host che ha accesso ai nodi di database SAP HANA.
