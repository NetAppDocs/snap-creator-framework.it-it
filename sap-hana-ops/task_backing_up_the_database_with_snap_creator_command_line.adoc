= Backup del database con la riga di comando di Snap Creator
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
È anche possibile eseguire il backup del database utilizzando la riga di comando di Snap Creator.

. Per eseguire il backup del database, eseguire il seguente comando.
+
[listing]
----
mgmtsrv01:~ # /opt/NetApp/Snap_Creator_Framework_411/scServer4.1.1/snapcreator --server
localhost --port 8443 --user scadmin --passwd scadmin --profile HANA_profile_ANA --config
ANA_database_backup --action backup --policy daily --verbose
